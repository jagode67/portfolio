SOT

SOT es una aplicación web diseñada para enseñar los pasos que hay que seguir a la hora de crear una aplicación web de principio a fin. Se muestra como está creada la base de datos, las API's necesarias y los elementos del front necesarios.

Esta aplicación integra diferentes tecnologías que permiten una comprensión completa del proceso de desarrollo web.

  • Gestión de Base de Datos
  • Integración con la API REST
  • Conexión con WhatsApp
  • Visualización de Google Maps con geolocalización
  • Integración con Inteligencia Artificial
